1. **dashboard.png** — Full window with a result (e.g. Leo / Fire)
2. **form.png** — Sidebar with month/day controls
3. **empty-state.png** — Optional, before first calculation

## How to capture (Windows)

1. Run the app: `python main.py` from the project root
2. Enter a birth date and click **Calculate sign**
3. Press **Win + Shift + S** (Snipping Tool) or **PrtScn**
4. Save files into this folder with the names above
